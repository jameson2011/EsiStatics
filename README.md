EsiStatics is a .NET Standard 2 library with  embedded static data from ESI. There are no database queries, API calls or JSON files to worry about: the data is encoded in plain old CLI code ready for use.

## What's included ##

This is a partial representation of ESI static data 

* Universe (regions, solar systems, celestials) 
* item types (groups, metagroups & market groups)
